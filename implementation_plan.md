# Shell Alias 可视化管理工具 (AliasGUI) - 优化版执行计划

本计划旨在构建一个极致简约、操作直观且体积轻量的 Shell 别名管理工具。

## 设计理念

- **极简主义**：界面清爽，功能聚焦。不堆砌无用功能。
- **操作直观**：所见即所得，支持行内编辑和即时搜索。
- **极致轻量**：采用 Vite + React 最小化依赖，Node.js 原生 IO。
- **安全保障**：每次修改前强制备份，支持一键还原，提供变更对比。

---

## 核心功能

1. **智能环境探测**：自动识别 macOS/Linux/Windows 及其 Shell 环境（zsh, bash, powershell）。
2. **可视化管理矩阵**：
   - 统一管理常规 `alias` 与带参数的 `function`。
   - **快速过滤**：顶部搜索框实时过滤别名列表。
   - **状态开关**：支持一键“禁用”别名而不必删除。
3. **安全变更流**：
   - **增量备份**：保存前生成带时间戳的备份文件。
   - **差异预览**：保存前告知用户哪些行将被修改。
   - **即时生效**：后台执行 `source` 等命令，无需手动重启终端。

---

## 技术架构

- **前端**：Vite + React + Vanilla CSS (极致响应速度)。
- **后端**：简易 Node.js 服务 (负责读写配置文件)。
- **交互**：本地运行 Web 服务器，浏览器操作。
